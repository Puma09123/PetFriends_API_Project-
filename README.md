# PetFriends API Project

Учебный проект по тестированию REST API сервиса PetFriends.

Проект выполнен в рамках заданий 24.2.4 и 24.7.2 курса SkillFactory.

## Описание проекта

В проекте используется готовая библиотека для работы с API PetFriends.
Два метода были реализованы самостоятельно в файле api.py на основе
официальной документации сервиса.

Также написаны 10 автоматических позитивных тестов, которые проверяют,
что API корректно обрабатывает запросы и возвращает статус-код 200.

## Структура проекта

PetFriends_API_Project/
│
├── api.py

## Используемые технологии

- Python 3
- requests
- unittest
- REST API

## Реализация методов API (api.py)

В файле api.py реализованы следующие методы:

- get_pets_list — получение списка всех питомцев
- add_new_pet — добавление нового питомца с фотографией

Методы используют HTTP-запросы GET и POST и возвращают объект response.

## Тестирование API

В файле test_pet_friends.py реализованы 10 позитивных тестов с использованием
библиотеки unittest.

Тесты проверяют:
- успешное получение списка питомцев;
- добавление нового питомца;
- корректный статус-код ответа (200);
- наличие ожидаемых полей в JSON-ответе;
- повторяемость запросов;
- корректный формат возвращаемых данных.

## Настройка токена

Для работы с API необходим токен авторизации.

В файле settings.py должен быть указан токен в следующем формате:

AUTH_TOKEN = "your_valid_token_here"

Токен в репозиторий не добавляется.

## Установка зависимостей

Установите зависимости командой:

pip install -r requirements.txt

## Запуск тестов

Для запуска тестов выполните команду:

python test_pet_friends.py
