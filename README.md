# PetFriends API Project (задание 24.7.2)

Учебный проект по тестированию REST API сервиса PetFriends в рамках курса SkillFactory.

## Описание проекта

В данном проекте реализована работа с REST API PetFriends.  
Реализованы недостающие методы API и написаны автоматические тесты для проверки
корректной работы сервиса при различных запросах.

Цель проекта:
- изучить работу с REST API;
- реализовать недостающие методы библиотеки;
- написать позитивные и негативные тесты;
- проверить корректность ответов и статус-кодов API.

## Структура проекта

PetFriends_API_Project/
│
├── api.py

## Используемые технологии

- Python 3
- requests
- unittest
- REST API

## Установка зависимостей

Для установки зависимостей выполните команду:

pip install -r requirements.txt

## Настройка проекта

В файле settings.py необходимо указать токен авторизации:

AUTH_TOKEN = "your_valid_token_here"

Токен в репозиторий не добавляется по соображениям безопасности.

## Описание файлов

api.py  
Содержит реализацию методов для работы с API PetFriends:
- получение списка питомцев;
- добавление нового питомца.

settings.py  
Содержит настройки проекта и токен авторизации.

test_pet_friends.py  
Содержит 10 автотестов для проверки работы REST API:
- позитивные проверки;
- проверки статус-кода 200;
- проверки структуры ответов;
- проверки повторяемости запросов.

requirements.txt  
Список библиотек, необходимых для работы проекта.

## Запуск тестов

Для запуска тестов выполните команду:

python test_pet_friends.py
